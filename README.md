# ChromaAvant - 前卫色彩实验室

> "色彩是视觉的语言，像素是创意的起点"
> 一个基于图像分析的智能色彩调色板提取工具

## 📖 项目简介

ChromaAvant（前卫色彩实验室）是一个创新的色彩设计工具，通过先进的像素分析算法从用户上传的图像中提取专业的色彩调色板。帮助设计师、艺术家和创意工作者从现有设计中汲取灵感，提升创作效率和色彩搭配水平。

## ✨ 主要功能

### 🎨 核心功能

- **智能色彩提取** - 基于像素分析算法提取图像中的主要色彩
- **专业调色板** - 自动生成5色调色板，包含RGB/HSL信息
- **色彩和谐度分析** - 分析色彩关系的和谐程度（互补、类似、三元等）
- **对比色建议** - 智能推荐与主色调形成良好对比的辅助色彩
- **一键复制** - 快速复制HEX、RGB、HSL色彩代码到剪贴板
- **响应式设计** - 完美适配桌面和移动设备

### 📊 功能特性

- 实时色彩预览和对比
- 精确的色彩占比统计
- 优雅的动画过渡效果
- 现代化的暗色系用户界面设计

## 📦 安装与运行

### 环境要求

## 💡 使用说明

### 1. 上传图像
   - 点击或拖拽图像文件到上传区域
   - 支持格式：JPG、PNG、WebP
   - 最大文件大小：10MB

### 2. 色彩分析
   - 系统自动分析图像中的主要色彩
   - 提取前5个最突出的颜色
   - 显示每个颜色的HEX、RGB、HSL值
   - 提供色彩占比统计

### 3. 色彩和谐度
   - 分析调色板的色彩关系
   - 识别互补色、类似色、三元色等和谐类型
   - 提供专业的色彩理论指导

### 4. 对比色建议
   - 基于主色调智能推荐对比色
   - 确保良好的视觉对比度和可读性
   - 一键复制建议的对比色代码

### 5. 色彩复制
   - 点击任意色彩样本进行复制
   - 支持HEX格式复制到剪贴板
   - 复制成功后会有视觉反馈

## 🔬 色彩分析技术

### 像素分析算法

ChromaAvant采用先进的像素级色彩分析技术：

1. **色彩量化** - 将图像像素进行色彩量化处理，合并相似颜色
2. **频率统计** - 统计各类色彩出现的频率，提取主要色彩
3. **和谐度分析** - 基于色彩理论分析调色板的和谐程度
4. **对比色计算** - 根据主色调的亮度智能推荐对比色

### 技术优势

- **精确分析** - 基于实际像素数据，确保色彩提取的准确性
- **实时处理** - 所有分析在浏览器本地完成，无需等待网络请求
- **隐私保护** - 图像文件不会上传到任何服务器，完全本地处理

## 💾 数据处理

ChromaAvant注重用户隐私和数据安全：

- **本地处理** - 所有图像分析在浏览器本地完成
- **无数据上传** - 图像文件不会发送到任何外部服务器
- **临时存储** - 仅在分析过程中临时使用Canvas API处理图像
- **无持久化** - 不保存用户的图像文件或分析历史
